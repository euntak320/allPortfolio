e-L/C

電子信用状とは、ペーパーでの文書に基 盤をおいた伝統的な信用状とは異なり、 電子的な方式で管理される信用状であ り、通知から輸出代金の買取に及ぶ全て のプロセスを電子化し、体系的に管管理 するシステムのことをいいます。

e-B/L

e-B/L Serviceは、運送会社から電子で B/Lを受信し、その所有権をTitle Registryに登録、原本は電子貿易文書保管所(uTrade Document Repository)に保管されます。e-B/L ServiceはB/Lを電子的に流通するための基盤を提供します。

e-NEGO

『e-Nego システム』は、Nego 関連文書の電子化及び Nego 書類の自動化を通じて輸出会社が関係機関及び銀行を訪問せず、uTradeHubを通じてNego業務をOne-Stopで処理することで輸出競争力を確保できるシステムです。

e-TRADE
Document Repository

貿易文書に対する組織的保管、管理及び原本保証により、電子文書の流通をサポートします。企業にとっては、ペーパー文書を提出せず、原本であることが保証された電子文書を自由に流通することができます。
