Introduction
迅速な貨物処理のための船社、航空者、フォワーダーなど輸出入物流会社向けのサービスです。

輸出入物流会社の業務機能の強化及び効率性を増大させるために物流会社の業務プロセスと自動で連携、時間と場所の制約を受けずにインターネットで保税貨物の処理業務を遂行できる輸出入貨物の統合業務処理サービスであり、積荷目録集合システム(MFCS)を使った貨物情報サービスなどを提供します。
