uTradeHub FEATURED SERVICE
e-L/C

电子信用证不同于过去基于纸张文件的信用证，是指以电算方式进行管理的信用证。 e-L/C Service是进行电算处理并系统地管理从通知到收汇的一系列过程的系统。

e-B/L

e-B/L Service通过电子系统从货运公司收到提单，在所有权注册中心（Title Registry）注册其物权凭证，正本储存在电子贸易文件存管中心(uTrade Document Repository)。e-B/L Service为提单流通的电子化奠定了基础。

e-NEGO

e-Nego系统”实现协商相关文件的电子化及协商文件的自动化，出口商不必亲自访问相关机构及银行，就可以通过uTradeHub，一站式处理贸易业务，提升出口竞争力。

e-TRADE
Document Repository

无须现有的书面信用证 通过对贸易资料的系统保管、管理及正本保证，支持电子资料的流通。从此企业在不提供书面资料的情况下可通过电子贸易资料保管所自由地流通经过正本保证的电子资料。
