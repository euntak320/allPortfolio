简介
为船运公司、航空公司、货代等进出口物流公司迅速地处理货物而提供的服务。

uLogis作为综合处理进出口货运业务的服务系统，为加强进出口物流公司业务功能，提高效率，可自动连接物流公司的业务流程。uLogis不受时间和地点的限制，通过网络办理保税货物业务，并通过集中托运系统（MFCS）,提供货运信息服务。
