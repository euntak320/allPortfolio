e-B/L 服务
简介

装船单虽然实现了部分电子化，但作为有价证券，提单的电子化尚未得以实现，成为阻碍电子贸易发展的绊脚石。e-B/L Service通过电子系统从货运公司收到提单，在所有权注册中心（Title Registry）注册其物权凭证，正本储存在电子贸易文件存管中心(uTrade Document Repository)。e-B/L Service为提单流通的电子化奠定了基础。

主要功能
装船单(Request Instruction)

装船单是出口商输入商业发票、装箱单、出口许可证、集装箱等信息，传送给货运公司要求装船的单据。

B/L查询 (B/L Inquiry)

e-B/L查询可分为e-B/L查询与Check e-B/L查询。e-B/L提供注册权限的详细内容查询、打印正本、打印复印本等功能。

修改e-B/L

修改e-B/L服务是修改从货运公司收到的e-B/L，或在发生错误时使用的功能。

e-B/L的转让申请

e-B/L转让申请”服务是代办e-B/L转让申请的功能，协助贸易商将自己所拥有的提单转让给另外贸易商。

接受提单的受让r

另外贸易商申请转让时，决定接受或拒绝提单的转让。

提单物权凭证注册内容查询

查询记录在所有权注册中心（Title Registry）的提单物权凭证注册、转让等变更事项

预期效果
  • · 减少因贸易商提单的物理流通而发生的时间、成本、业务的负担。
  • · 网上办理提单的签发及流通，缩短承兑时间。
  • · 通过e-Nego，实现出口贷款的早期贴现。
