预期效果
电子贸易一条龙服务

以uTradeHub整合现有的电子贸易服务与进出口贸易相关机构服务。

按照贸易流程，提供业务支持

UTradeHub作为一种基于BPM(Business Process Management)的智能型服务，分为出口、进口、本地业务，根据不同的贸易流程，自动生成并进行下一步工作。

再利用信息

信息一经输入即储存在uTradeHub里，其他文件也可以自动取用之，因此无需重复输入相同信息。

简便地收发文件

只要进行点击，就可以将储存在uTradeHub电子文件存管中心的文件递交到银行、物流公司、税局等。

缩短工作时间及节省成本

省略一切离线程序，一律在uTradeHub上办理业务，缩短了工作时间，节省了成本。

按照不同用户，提供定身服务

每个人可以设定不同的菜单及服务，在个人操作环境下开展业务。

提供多元化的使用环境

根据各业者的IT环境，采取如门户网站、解决方案、网络服务等不同的方式，使用电子贸易服务。

安全的交易

只有对信息及文件持有使用权的用户，才能接近并使用相关资料，确保交易的安全性，避免伪造、变造及金融事故的发生。
