简介

uTradeHub是利用网络等尖端通信技术，与现有贸易程序上的相关机构有效地连接的新概念国家电子贸易平台，是无间断办理从营销到结汇的一系列贸易流程。uTradeHub基础设备由电子文件中介系统、电子文件存管中心、所有权注册中心（Title Registry）的链接界面组成。
