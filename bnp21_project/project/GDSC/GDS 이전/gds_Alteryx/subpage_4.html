ALTERYX PROMOTE 분석을 위한 협업 및 데이터 탐색 플랫폼

예측 모델을 구축하는 것을 어려운 일입니다. 또한 그 모델을 디플로이 하는 것도 마찬가지 입니다. 예측 모델의 배치 단계에 있어 오류 발생이 높고, 노동 집약적이며, 여러 단계를 거쳐야 합니다. 또한 구축에 있어 수개월의 기간과 수 억원의 자금이 소요됩니다.
Alteryx Promote는 예측 모델 구축, 배치 및 관리 그리고 의사결정 API를 제공하는 엔드 투 엔드 데이터 사이언스 시스템 입니다. 이를 통해 데이터 과학자 및 분석 팀은 커스텀 배포 코드를 작성하지 않고도 예측 모델 생산에 보다 빠르고 안정적으로 구축, 관리 및 배포 할 수 있습니다.

DEPLOY 프로덕션 환경에 더 많은 분석 모델을보다 빠르고 더 적은 시간과 리소스로 유연하게 배포 할 수 있습니다. Alteryx Promote는 모든 실시간 요구에 따라 어느 때보 다 쉽게 모델을 배치합니다.
· Code-Friendly 모델 배포 데이터 과학자는 기존의 라이브러리를 사용할 수 있는 선택의 자유를 유지하면서 많은 시간 소모적 인 재코딩 작업 없이 커스텀 R 및 Python 모델을 신속하게 배포 할 수 있습니다. · Code-Free 모델 배포 일반 데이터 과학자를 위해 Promote는 모든 기능을 턴키로 제공하면서 자동화된 솔루션인 Alteryx Designer를 통해 고급 예측 분석 및 머신러닝을 코딩없이 쉽게 배포할 수 있게 합니다. · Application Ready 표준 REST API 및 Java, Python, Ruby, PHP, Saleforce, Node.JS를 위한 코드 스니펫(snippet)제공으로 쉽게 여러분의 실시간 어플리케이션에 모델을 임베디드 할 수 있습니다.

MANAGE 지속적으로 새로운 모델을 배포하고, 현재 모델을 개선하고 또한 성능이나 가용성을 높이기 위해 쉽게 확장할 수 있도록 단일 배포 프로세스로 디자인되어 있어 비즈니스 성능을 향상 시킵니다.
· Model Management 운영 어플리케이션에서 실행되는 기존 모델에 영향을 주지 않고, 단위 테스트를 포함하여 통합 배치 과정에서 부터 운영 전개 과정까지 보다 쉽게 관리할 수 있습니다. · Model Versioning Promote의 버전 컨트롤 및 추적 기능을 사용하여 여러 버전을 안정적으로 관리하고 원활하게 롤백하여 팀 전체의 시간과 노력을 절감할 수 있습니다. · Maximize Availability 확장 및 장애복구를 위해 신속한 모델 복제 구성으로 모델에 의존된 여러분의 미션 커리티컬한 응용 프로그램의 가용성을 극대화할 수 있도록 보장합니다.
MANAGE 이미지
MONITOR 여러분의 예측 모델의 상태, 성능, 가용성 등을 안전하게 모니터하여 여러분의 분석적 혁신이 업무에 이익 극대화를 가져 줄 수 있도록 보장합니다.
· Model Performance 실시간 대시보드와 빌트인 된 시각적 분석기능을 사용하여 모델 리소스 사용, 응답 시간, 요청 기록 및 성능 기록을 모델별로 파악할 수 있습니다. · System Health Overview 단일의 중앙집중식 뷰어에서 모든 관련 서버의 CPU, 메모리 및 디스크 사용 현황 등의 시스템 리소스를 철저히 모니터링 합니다.
