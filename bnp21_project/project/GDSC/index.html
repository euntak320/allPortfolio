유영민 과기정통부 장관, 데이터전문기업 현장방문

유영민 과학기술정보통신부 장관은 6일 CJ올리브네트웍스를 찾아 데이터 전문기업들과 '데이터 산업 혁신성장 방안 간담회'를 개최했다. 이번 간담회는 4차 산업혁명 시대 핵심 자본인 데이터 기반 혁신을 촉진하고 데이터 산업계의 애로사항을 청취하기 위해 마련됐다. 올들어 유영민 장관이 데이터 전문기업을 현장방문한 것은 이번이 처음이다.
