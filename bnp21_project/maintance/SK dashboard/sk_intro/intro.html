Sl Client 사용 유의 사항
  • 1. JAVA versrion: Sl Client는 JAVA Version 8에 최적화되어 있습니다. 기타 버전에서는 일부 기능이 정상적으로 작동하지 않을 수 있습니다.
  • JAVA Version 8는 Orcale JAVA 홈페이지에서 다운로드 받아 설치하신 후 실행하시기 바랍니다.
  • 2. 최초 실행시 보안 경고창이 표시될 수 있으며 이 때 "예"를 선택해야만 실행이 가능합니다.
  • 3. Sl server에 login 하기 위해서는 User ID를 등록해야 됩니다. 필요시 Sl server 관리자에게 요청하십시요.
Troubleshooting Guide
  • 1. ESM client 실행 전에 다른 응용프로그램을 종료하여 가용메모리르 확보하기를 권장합니다.
  • 2. ESM client 실행이 실패하거나, 최신 PKG로 동작하기 않는 것처럼 보이는 경우에는 아래와 같은 방법으로 PC에 저장된 JAVA Cache를 삭제한 후 실행해 보시기 바랍니다.
